The graph API basically represent the agent workflows and interactions as a graph.

Part of graph api:
    - State:
        - Sender
        - Recipient
        - Body
        - Status
    - Nodes
        - 
        - def nodeA(state)
    - Edges
        - How the flow of the AI agent is going to go for the particular operation is supposed.
        - def edge(state)


Steps to create Graph:
    - Define your state (StateGraph)
    - Create your nodes
    - Connect your nodes with edges
    - Compile your graph
    - Run your graph

#### Start node and End node is essistial for graph
